# โรงแรมภูสักธาร — ระบบโหวตชุดอาหาร

โปรเจกต์นี้เชื่อม Firebase Project `phusakthanvote` แล้ว

## ไฟล์
- index.html
- style.css
- app.js
- firebase.js
- firestore.rules
- netlify.toml

## วิธีอัปโหลดเข้า GitHub บน iPhone
1. แตกไฟล์ ZIP ในแอป Files
2. เข้า Repository `phusakthan-food-vote`
3. กดปุ่ม `...` ข้างปุ่ม Code
4. เลือก `Upload files`
5. อัปโหลดไฟล์ทั้งหมดจากโฟลเดอร์ที่แตก ZIP
6. กด Commit changes

## นำขึ้น Netlify
1. เข้า https://app.netlify.com
2. สมัคร/เข้าสู่ระบบด้วย GitHub
3. เลือก Add new site > Import an existing project
4. เลือก GitHub และ Repository `phusakthan-food-vote`
5. Build command ปล่อยว่าง
6. Publish directory ใส่ `.`
7. กด Deploy site
